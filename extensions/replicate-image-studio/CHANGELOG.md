# Changelog — Replicate Image Studio

## [0.1.1] — 2026-02-27

### Changed
- Display name updated to **CX Replicate Image Studio** for consistent CX brand identity

## [0.1.0] — 2026-02-24

### Added
- Initial release
- `Replicate: Generate Image` — open the generation panel and produce AI images from a prompt
- `Replicate: Batch Generate` — queue multiple prompts from a JSON/YAML list file
- `Replicate: Show History` — browse generated images from the current session
- `Replicate: Configure Model` — choose from supported Replicate models (Flux, SDXL, Ideogram)
- `Replicate: Insert Image to Document` — embed the selected image into the active editor
- Configurable Replicate API token, model, output directory, and default dimensions
- Real-time progress indicator while predictions run
