# Changelog — Mermaid Diagram Pro

## [0.1.1] — 2026-02-27

### Changed
- Display name updated to **CX Mermaid Diagram Pro** for consistent CX brand identity

## [0.1.0] — 2026-02-24

### Added
- Initial release
- `Mermaid Pro: Preview Diagram` — live side-by-side preview of Mermaid diagrams
- `Mermaid Pro: Export as PNG` — render and save current diagram as PNG
- `Mermaid Pro: Export as SVG` — export diagram as an editable SVG file
- `Mermaid Pro: Insert Diagram Template` — quick-pick a starter diagram type
- Auto-preview on save for `.mmd` and fenced Mermaid blocks in Markdown
- Configurable theme (default, dark, forest, neutral)
- Error highlighting with inline diagnostics for invalid Mermaid syntax
